# Legacy Bullet Cluster scripts

These scripts were used during the exploratory phase of the
cluster-merger analysis (sessions up to Round 42). They are
superseded by `fig_cluster_merger_suite.py` (repo root), which
generates both publication figures for the current paper.

## Superseded files

| File | Replaced by |
|---|---|
| `cluster_phi_solver_2d.py` | `fig_cluster_merger_suite.py` (2D closure) |
| `bullet_3d_phi_solver.py` | `fig_cluster_merger_suite.py` (3D FFT-Poisson) |
| `calc_cluster_lensing.py` | `fig_cluster_merger_suite.py` |
| `bullet_cluster_ect_test.py` | Exploratory falsification tests; superseded |
| `fig_bullet_ect_maps.py` | `fig_cluster_merger_suite.py` panel (a–c) |
| `fig_bullet_ect_profiles.py` | `fig_cluster_merger_suite.py` panel (d) |
| `fig_bullet_ect_scan.py` | Parameter scan; superseded |
| `bullet_3d_phi_maps.png` | `fig_bullet_main.png` |
| `bullet_3d_phi_profiles.png` | `fig_cluster_suite_budget.png` panels (e–h) |

## Current figures (repo root)
